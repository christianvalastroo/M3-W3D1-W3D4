# 🎓 Geeks UI – Bootstrap Course Platform

## 📌 Descrizione

Replica di una piattaforma e-learning realizzata con **Bootstrap 5.3**.  
Il progetto include layout responsive, sezioni corsi con card dinamiche e animazioni hover personalizzate.

---

## 🚀 Tecnologie utilizzate

- 🧱 HTML5  
- 🎨 CSS3  
- 🅱️ Bootstrap 5.3  
- ⭐ Bootstrap Icons  
- 🔤 Google Fonts (Inter)

---

## 📱 Responsive Design

Il progetto è completamente responsive:

- 📲 **Mobile**
  - Le card dei corsi diventano un **carousel**
- 💻 **Tablet / Desktop**
  - Layout a **griglia** (`row-cols-md-3` / `row-cols-lg-4`)

---

## ✨ Funzionalità principali

- 🧭 Navbar con dropdown
- 🎯 Hero section con call-to-action
- 📊 Sezione dettagli servizi
- 📚 3 sezioni corsi:
  - Recommended
  - Most Popular
  - Trending
- 🎴 Hover animation sulle card
- 🔖 Bottone “Salva” con animazione hover personalizzata
- 🦶 Footer responsive

---

## 🎨 Animazioni implementate

### 🃏 Card hover
- Leggera elevazione (`translateY`)
- Ombra morbida
- Transizione fluida

### 🔖 Bottone Salva
- Sostituita icona semplice con `<button>`
- Animazione hover con effetto visivo
- Transizioni fluide e leggere
